A role-based Crime Investigation Management System built during my ReadyNest Summer Internship. The project manages the complete investigation process, from FIR registration to final case closure, with separate dashboards for different officer roles.

<img width="1536" height="698" alt="Screenshot 2026-07-22 182636" src="https://github.com/user-attachments/assets/9204778f-cc53-4ddb-b56c-4c4b207b1a67" />


## Features

- Role-Based Authentication
- FIR Registration
- Case Assignment
- Digital Evidence Upload
- Investigation Timeline
- Forensic Workflow
- Superintendent Review
- Chargesheet Approval
- Audit Logs
- Dashboard Analytics
- Responsive UI

## Tech Stack

**Frontend:** HTML, CSS, JavaScript

**Backend:** Node.js, Express.js

**Database:** PostgreSQL, Prisma ORM

**Other:** JWT, Cloudinary, Multer, Socket.io

---

## Workflow

```text
FIR Registration
      ↓
Case Creation
      ↓
Assign Sub Inspector
      ↓
Evidence Collection
      ↓
Forensic Analysis
      ↓
Superintendent Review
      ↓
Chargesheet Approval
      ↓
Case Closed
```
